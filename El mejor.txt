¿Cuál es el mejor IDE para C y C++?
No existe el IDE perfecto. Además hay muchas opciones.
En Windows la opción típica es Microsoft Visual Studio[1]. Además de muchos otros lenguajes, soporta C++, y el compilador y enlazador soportan C. Dispones de Visual Studio Code, libre y gratuito. Además, las últimas versiones funcionan en Linux y Mac OS X.
Entre otros estan : netbeans, code blocks, entre otros.
